# Bash scripts

*Run the example policies to get familiar with the simulation*

1. Change to the container directory
```
user@host:~$ cd ~/ai_industry_challenge_deep_dive/bash/container
```

2. Run the bash script for the `/aic_eval` container in terminal 1

```
user@host:~/ai_industry_challenge_deep_dive/bash/container$ ./aic_eval_container.bash
```

3. Run the bash script for the example policy *WaveArm* in a new terminal

```
user@host~$ cd ~/ai_industry_challenge_deep_dive/bash/example_policies
user@host:~/ai_industry_challenge_deep_dive/bash/example_policies$ ./wave_arm.bash
```

> [!NOTE]
> If you want to run the **CheatCode** policy you need to rerun the container with a ground true of **true**. Change this in the `aic_eval_container.bash` file
